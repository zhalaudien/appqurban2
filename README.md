# Aplikasi qurban pusat 7

Ini adalah pengembangan aplikasi web untuk memudahkan dalam managemen data dipantia qurban pusat 7 berbasiskan codeigniter 4.

## Cara Instalasi

`git https://github.com/zhalaudien/appqurban2.git`
`cd appqurban2`
`composer update`
`php spark serve`

## Fitur Fitur

- User login
- Data panitia
- Data cabang
- Data hewan qurban
- Data pengiriman
- Data permintaan
